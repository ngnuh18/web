# Web Project

Đây là dự án web của team.

## Link truy cập
- **Production**: https://thepetvietnam.vercel.app/
- **Local development**: http://localhost:3000/ (chạy `npm run dev` để khởi động)

*Lưu ý: Link localhost chỉ hoạt động trên máy local. Team members truy cập production link.*

## Cách chạy
1. Clone repo: `git clone https://github.com/ngnuh18/web.git`
2. Install dependencies: `npm install`
3. Run: `npm run dev`

## Tech stack
- Next.js
- Tailwind CSS
- TypeScript
